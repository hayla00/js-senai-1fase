vamos ver alguns atalhos legais no VSCode

alt+z:quebra automática de linha

alt+setas cima/baixo: move a seleção para cima/para baixo no arquivo. se nada estiver selecionado, movimenta a linha em que o cursor está

ctrl+espaço: chama o autocompletar de contexto

shift+alt+setas cima/baixo: copia

ctrl+; : comentário/descomentário
